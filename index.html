Nozomi UI library

theme-aware roblox UI with a dynamic island heart.

Nozomi started as my playground for glassmorphism on Roblox. now it's a full toolkit: sliders, dropdowns, confirm dialogs, and that signature floating island that keeps stats front-and-center.

feature highlights

  • theme presets with live tween updates (“Sakura Dark”, “Nord”, “Soft Mint”, and more)
  • asset manager that caches Lucide icons via getcustomasset
